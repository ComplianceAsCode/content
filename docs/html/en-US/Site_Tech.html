Site Tech

Technology used to create this site

This site is constructed with Publican, a free and open-source publishing tool developed by Red Hat, Inc.
Publican takes content authored in DocBook XML and transforms it into a variety of formats such as HTML, PDF, and EPUB. It can also build RPM packages to ship the content and can build documentation websites to publish the content on the web.
Publican is included in the Fedora and Ubuntu distributions of Linux. Run yum install publican on Fedora or apt-get install publican on Ubuntu to install Publican on your system. An installer for Windows operating systems is also available.
